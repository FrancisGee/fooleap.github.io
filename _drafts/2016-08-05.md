Google Earth 的卫星图一直比国内地图所用的卫星图要新一些，然而 Google 早已不在墙内，中文版的谷歌地图虽能访问，但路网数据各方面并国内地图的完善。我就想在使用高德地图 API ，加载谷歌的卫星地图图层，并使用高德的路网。

这个需求用高德地图 API 很好实现，只需叠加谷歌的卫星图层再叠加高德地图的路网图层便可[[1]][1]。
